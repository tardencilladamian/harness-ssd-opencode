# Requirements - F-00 Project Foundation

## R1

WHEN an AI agent starts a session, THE SYSTEM SHALL provide a clear repository navigation map through `AGENTS.md`.

## R2

WHEN a feature is selected for work, THE SYSTEM SHALL provide a traceable specification containing requirements, design, and tasks.

## R3

WHEN a feature is implemented, THE SYSTEM SHALL provide automated or documented verification evidence.

## R4

WHEN a feature is closed, THE SYSTEM SHALL record progress and review evidence.

## Out of Scope

- Product-specific features.
- Stack-specific implementation.
- Deployment automation.
