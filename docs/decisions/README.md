# Decisions

This folder stores approved technical and product decisions.

Use one file per decision:

```text
YYYY-MM-DD-short-title.md
```

Decision template:

```markdown
# Decision - Short Title

## Date

YYYY-MM-DD

## Status

approved | superseded | rejected

## Context

Why this decision was needed.

## Decision

What was decided.

## Consequences

Expected tradeoffs and follow-up work.
```
